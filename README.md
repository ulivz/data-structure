## Usage

```bash
yarn add dast -S
```

```js
import {
  Stack,
  Queue,
  LinkedList,
  LinkedNode,
  DoublyLinkedList
  // ...
} from 'dast'
```

or use it with CDN:

```html
<script src="https://unpkg.com/dast@latest/dist/data-structure.min.js"></script>
```

```js
const {
  Stack,
  Queue,
  LinkedList,
  LinkedNode,
  DoublyLinkedList
  // ...
} = window.dast
```
